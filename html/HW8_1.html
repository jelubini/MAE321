Problem 1:

A machine modeled as a linear spring-mass-damper system, is driven at resonance (). Design a damper (choose a value of c) so that the maximum deflection at steady state is 0.05 m. The machine is modeled as having a stiffness of 1700 N/m, and the excitation force has a magnitude of 150 N.

Find: Design System,

Known

frequencyNatural = 2.5;              % rad/s
frequencyDriving = frequencyNatural; % rad/s
amplitudeSS      = 0.05;             % m
stiffness        = 1700;             % N/m
forceDriving     = 150;              % N

Calculations

Steady State Amplitude:

forceDrivingMassStd = forceDriving / mass;
